# .codex

Codex-specific project config lives here.

Behavioral instructions live in root `AGENTS.md` and app-local `AGENTS.md`
files. This directory is config-only.

Files:

- `config.toml` - Codex-style project MCP server config.
- `mcp.toml` - MCP TOML mirror for tools that probe this filename.

Do not add command prompt files, workflow runbooks, API keys, tokens, or other
secrets here.
